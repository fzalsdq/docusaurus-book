# Git Commands Used

Here is a list of Git commands executed during our session:

## Checking Git Credentials
- `git config user.name`
- `git config user.email`

## Setting up Remote and Initial Push
- `git remote -v`
- `git remote add origin https://github.com/fzalsdq/docusaurus-book.git`
- `git remote -v`
- `git add .`
- `git commit -m "Initial commit"`
- `git push -u origin master`
